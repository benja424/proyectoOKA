# Sistema de Gestión de Traslados y Documentación Clínica

## Equipo

ROKA

---

## Descripción

Este proyecto consiste en el desarrollo de un sistema web orientado a la gestión de traslados en ambulancia y al acceso a documentación clínica mediante códigos QR.

El sistema está diseñado para ser utilizado tanto por personal administrativo como por pacientes, ofreciendo una solución eficiente, accesible y adaptable a distintos dispositivos.

---

## Módulos del Sistema

### Módulo de Documentación Clínica

* Acceso a documentos mediante códigos QR
* Visualización de información desde dispositivos móviles (mobile-first)
* Formulario de encuesta de satisfacción para pacientes

### Módulo de Ambulancias

* Panel de gestión de traslados
* Registro de nuevas solicitudes de traslado
* Seguimiento del estado de los traslados

---

## Tecnologías Utilizadas

* Backend: PHP
* Base de datos: MySQL / MariaDB
* Frontend: Bootstrap 5.3
* Control de versiones: Git y GitHub
* Entorno de desarrollo: Visual Studio Code

---

## Requisitos del Sistema

* XAMPP (Apache y MySQL)
* PHP 8 o superior
* Navegador web actualizado
* Visual Studio Code (opcional, recomendado)

---

## Instalación

1. Clonar el repositorio:

git clone https://github.com/benja424/proyectoOKA.git

2. Ubicar el proyecto en la carpeta:

C:\xampp\htdocs\

3. Iniciar los servicios Apache y MySQL desde el panel de XAMPP

4. Crear la base de datos desde phpMyAdmin:

http://localhost/phpmyadmin

5. Importar el archivo SQL (si está disponible en /database)

6. Acceder al sistema desde el navegador:

http://localhost/proyectoOKA

---

## Control de Versiones

Se utilizan las siguientes convenciones de commits:

* init: creación inicial del proyecto
* feat: nueva funcionalidad
* fix: corrección de errores
* docs: documentación
* style: cambios de formato

Ejemplos:

* feat: alta de traslado
* fix: validación de formulario
* docs: actualización de README

---

## Configuración del Entorno

El sistema está preparado para ejecutarse en un entorno local utilizando XAMPP.

Pasos básicos:

* Instalar XAMPP
* Iniciar Apache y MySQL
* Clonar el repositorio
* Colocar el proyecto en htdocs
* Configurar la base de datos
* Ejecutar desde el navegador
